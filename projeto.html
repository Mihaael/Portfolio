Viva o Antigo

É um projeto que tem como objetivo trazer o sentimento de pertencimento para as pessoas no recife antigo, o projeto envolve uma câmera fixada numa caixa, onde o botão de tirar foto é acionado através de uma tela. Se a pessoa desejar tirar uma foto, será iniciada uma contagem para ela se posicionar. No fim da contagem, o Arduino emitirá uma sinalização para o servo motor que realizará o clique para foto.
