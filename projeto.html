Viva o Antigo

É um projeto que tem como objetivo trazer o sentimento de pertencimento para as pessoas no recife antigo, o projeto envolve uma camera fixada numa caixa, onde o botão de tirar foto é acionado através de uma tela, se a pessoa desejar tirar uma foto ira ser iniciado uma contagem para ela se posicionar, no fim da contagem o arduino ira emitir uma sinalização para o servo motor que ira realizar o clique para foto.
